# AI Fake News Detector

An intelligent Fake News Detection Web Application built using Flask, Natural Language Processing (NLP), and Generative AI.  
The system analyzes news statements and determines whether they are Real, Fake, or Uncertain by combining multiple verification sources.

---

# Features

- Text-based Fake News Detection
- Sentence Embeddings (Sentence Transformers)
- Wikipedia & NewsAPI Verification
- Google Gemini (Generative AI) Context Analysis
- Google Fact Check API Integration
- Confidence Score with Visual Indicator
- Session-based Analysis History
- Modern Responsive UI (HTML + CSS)
